## vnpy快速入门

####  安装
```
docker pull efreeway/ubuntu-vnpy-novnc:vnpy-1.9.1  #下载  
docker tag efreeway/ubuntu-vnpy-novnc:vnpy-1.9.1  vnpynovnc  #换一个好记的名字  
docker run -it -p 6080:80 vnpynovnc  #运行容器 
```
